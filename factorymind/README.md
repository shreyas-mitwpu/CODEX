# FactoryMind

AI Agent Factory Management System for small Indian auto parts manufacturers. 
Built for the Codex Community Hackathon.

## Tech Stack
- React (Vite)
- Tailwind CSS (via CDN)
- NO Backend, fully runs in the browser.

## Deployment (Vercel)
This project is configured for 1-click Vercel deployment. 
Simply push to GitHub and import the repository into Vercel.

## Local Setup
```bash
npm install
npm run dev
```

## Hackathon Demo Script
1. Open the app in your browser.
2. Say: "Hi judges. This is FactoryMind. We're solving for Raju Auto Parts in Pune."
3. Click "⚡ Run FactoryMind Agents"
4. Say: "4 Codex agents orchestrate here in 28 seconds."
5. Read off the agents as they stream:
   - "Inventory finds critical Steel Rod shortage..."
   - "Production reorders priorities..."
   - "Maintenance flags Machine 2 risk..."
   - "Forecast projects +38% demand..."
6. The dashboard will populate. Say: "₹57,000 saved this week. ROI: 5,600%."
